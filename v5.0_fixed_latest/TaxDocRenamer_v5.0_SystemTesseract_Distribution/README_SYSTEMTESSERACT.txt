# 税務書類リネームシステム v5.0 SystemTesseract版

このディレクトリにはシステムTesseract依存版の実行ファイルが含まれています。

## 🚀 使用方法

### 1. Tesseractの事前インストール（必須）
このバージョンはシステムにインストールされたTesseractに依存します。

**Windows:**
1. https://github.com/UB-Mannheim/tesseract/wiki からTesseractをダウンロード
2. インストーラーを実行
3. 日本語言語パック (jpn.traineddata) を含めてインストール

**確認方法:**
```cmd
tesseract --version
```

### 2. アプリケーション起動
`TaxDocRenamer_v5.0_SystemTesseract.exe` をダブルクリック

### 3. 基本操作
1. PDFファイルをドラッグ&ドロップまたはファイル選択
2. 自治体情報・年月(YYMM)を入力
3. 分割・分類・リネーム実行

## ✨ 主な機能

### 分類・命名システム
- **XXXX_書類名_YYMM.pdf** 形式での自動命名
- **4桁番号体系**: 0000番台（国税）、1000番台（都道府県税）、2000番台（市町村税）、3000番台（消費税）等

### AND条件分類
- 複数キーワードの同時マッチング判定
- 受信通知・納付情報の高精度識別

### 自治体対応
- 最大5つの都道府県・市町村への提出対応
- 東京都優先配置（セット1必須）
- 連番による自動管理（1001, 1011, 1021...）

## 🎯 主要な命名例

### 国税（0000番台）
- `0001_法人税及び地方法人税申告書_2508.pdf`
- `0003_受信通知_2508.pdf`
- `0004_納付情報_2508.pdf`

### 都道府県税（1000番台）
- `1001_東京都_法人都道府県民税・事業税・特別法人事業税_2508.pdf`
- `1011_愛知県_法人都道府県民税・事業税・特別法人事業税_2508.pdf`

### 市町村税（2000番台）
- `2001_愛知県蒲郡市_法人市民税_2508.pdf`
- `2011_福岡県福岡市_法人市民税_2508.pdf`

### 消費税（3000番台）
- `3001_消費税及び地方消費税申告書_2508.pdf`
- `3003_受信通知_2508.pdf`
- `3004_納付情報_2508.pdf`

## 🔧 システム要件

- **OS**: Windows 10/11
- **Tesseract OCR**: インストール必須（日本語言語パック含む）
- **メモリ**: 4GB以上推奨
- **Python環境**: 不要（実行ファイル版）

## 🔧 トラブルシュート

### 「Tesseractが見つかりません」
**解決方法**: 
1. Tesseractが正しくインストールされているか確認
2. システムPATHにTesseractが含まれているか確認
3. 日本語言語パック（jpn.traineddata）がインストールされているか確認

### 「分類結果が期待と違う」
**解決方法**:
1. PDFのOCR精度を確認
2. 入力テキストに必要なキーワードが含まれているか確認
3. AND条件設定を見直し

### 起動に時間がかかる
**原因**: 大容量の実行ファイルのため初回起動は時間がかかる場合があります
**解決方法**: しばらくお待ちください
