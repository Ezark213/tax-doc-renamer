# インストール手順

## 必要環境
- Python 3.8以上
- PyMuPDF ライブラリ

## インストール手順
1. 必要ライブラリのインストール:
   pip install -r requirements.txt

2. システム起動:
   - Windows: start_system.bat をダブルクリック
   - コマンドライン: python main_v5.py

3. 使用方法:
   - v5.0モードにチェックを入れる
   - PDFフォルダを選択
   - リネーム実行

## テスト実行
python test_v5.py
